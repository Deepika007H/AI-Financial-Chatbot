## Project Overview
An AI-powered financial chatbot that analyzes 459 Indian companies 
including Reliance, TCS, HDFC and more using Python and Data Analytics.

## Skills Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Data Cleaning
- Data Visualization
- Chatbot Development

## Dataset
- 459 Indian companies
- Market Capitalization data
- Quarterly Sales data
- Source: Kaggle

## Project Steps
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization
5. AI Chatbot Development

## Key Insights
- Reliance Industries is largest company by Market Cap
- TCS has highest Market Cap among IT companies
- Dataset contains Large Cap, Mid Cap and Small Cap companies
